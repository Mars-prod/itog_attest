# Random Password Generator

Генератор случайных паролей с графическим интерфейсом на Python.

## Автор

**Сабирзянов Марсель Ильсурович**

## Описание проекта

Приложение для генерации надежных паролей с возможностью настройки параметров, сохранением истории генерации в JSON и удобным графическим интерфейсом на базе библиотеки Tkinter.

### Возможности кода

- **Настройка параметров пароля:**
  - Длина пароля (от 4 до 32 символов) с помощью ползунка
  - Выбор типов символов через чекбоксы:
    - Строчные буквы (a-z)
    - Заглавные буквы (A-Z)
    - Цифры (0-9)
    - Специальные символы (!@#$%^&* и др.)

- **Основные функции:**
  - Генерация случайного пароля по нажатию кнопки
  - Копирование пароля в буфер обмена
  - Автоматическое сохранение истории всех сгенерированных паролей
  - Загрузка истории из JSON файла при запуске
  - Очистка истории с подтверждением действия

- **Валидация и защита:**
  - Проверка минимальной (4 символа) и максимальной (32 символа) длины
  - Обязательный выбор хотя бы одного типа символов
  - Ограничение истории 50 последними записями

## Как работает история паролей

История паролей реализована следующим образом:

1. **Сохранение:** Каждый раз при нажатии кнопки "Сгенерировать пароль" запись автоматически добавляется в историю. Каждая запись содержит:
   - Дату и время генерации (формат: ГГГГ-ММ-ДД ЧЧ:ММ:СС)
   - Сгенерированный пароль
   - Длину пароля
   - Выбранные настройки (какие символы использовались)

2. **Хранение:** Все записи сохраняются в файл `password_history.json` в формате JSON. Файл создается автоматически в той же папке, где находится программа.

3. **Загрузка:** При запуске приложение автоматически загружает историю из JSON файла и отображает её в таблице.

4. **Отображение:** История показывается в виде таблицы с колонками:
   - Дата
   - Пароль
   - Длина
   - Настройки

5. **Ограничение размера:** Хранится только 50 последних записей, чтобы файл не разрастался бесконечно.

6. **Очистка:** Кнопка "Очистить историю" удаляет все записи (с подтверждением пользователя).

## Используемые библиотеки

| Библиотека | Версия | Назначение |
|------------|--------|-------------|
| tkinter | (встроенная) | Создание графического интерфейса |
| random | (встроенная) | Генерация случайных символов для пароля |
| string | (встроенная) | Доступ к наборам символов (буквы, цифры, спецсимволы) |
| json | (встроенная) | Сохранение и загрузка истории паролей |
| os | (встроенная) | Проверка существования файла истории |
| datetime | (встроенная) | Запись даты и времени генерации пароля |

Все библиотеки входят в стандартную поставку Python, дополнительная установка не требуется.

- Операционная система: Windows / Linux / macOS
